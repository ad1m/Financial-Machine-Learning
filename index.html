Financial Machine Learning

by Adam Lieberman

Meshing finance, mathematics, and computer science to build predictive quantitative financial systems

Introduction

Machine Learning (ML) is arguably one of the hottest and most exciting fields of study, making huge impacts and advancements across many industries and product lines. Machine learning systems are helping organizations accomplish complex tasks in a seamingly easy manner as these systems learn to understand, predict, adapt, and operate on thier own. They can injest large volumes of data at a given time, impossible for the human brain, learn and extract complex patterns from this data, and make inference and predictions that can adapt to changing environments. Companies and research institutes all around the world are leveraging machine learning such as

  • Amazon - Offering personalised recommendations based on previous purchases or activity.
  • Apple - Siri uses machine learning for voice recognition as part of the virtual assistant feature on the iPhone.
  • Facebook - Friends are automatically tagged in pictures through a machine learning system that was built for facial recognition.
  • Google - Google Maps analyzes the speed of traffic through locations and suggests alternative routes all through the use of machine learning.
  • Paypal - Customer data is analyzed through machine learning systems to determine fradulent transactions
  • Two Sigma - Algorithmic trading systems are employing machine learning tactics to trade stocks among other derivatives at high frequency rates.
  • Uber - Rider pickup times use machine learning systems to determine arrival times
  • Wealthfront - "Robo-Advisor" sytems calibrate financial portfolios to the goals and risk tolerance of the user for active portfolio management.

Machine learning is a technology that is disrupting every industry, especially finance. Hedge funds are using machine learning systems to build high frequency trading algorithms, wealth management firms are creating "Robo-Advisors" to optimize client portfolios, financial technology firms are selling news analysis software to better understand public sentiment relating to market direction, and many more. It is a very exciting time for machine learning and as it relates to finance, this technology will transform how everyone invests.

To understand how machine learning can be used in finance there are a few key fields that should be understood: Mathematics, Computer Science, and Finance. The goal of this book is to expose the reader to these three fields such that they can apply machine learning methods to build quantitative financial systems. To follow this book, a high level of experience is not needed in any of the above fields. We will navigate these fields and see how they work together so that we can apply machine learning in finance.
